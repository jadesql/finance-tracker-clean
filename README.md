# 💸 Finance Tracker (Streamlit App)

A simple personal finance dashboard built with Python, Streamlit, and SQLite.  
It helps track spending by category and visualise monthly expenses in a clean, fintech bank style interface.

---

## 📊 Features

- Add and view transactions
- Clean category system:
  - Food
  - Fuel
  - Shopping
  - Health
  - Eating out
  - Holidays
- Automatic spending summary
- Category breakdown dashboard
- Interactive bar chart (pink themed)
- Dark mode UI (Revolut-inspired)
- SQLite database storage

---

## 🧠 Tech Stack

- Python
- Streamlit
- Pandas
- SQLite
- Matplotlib
